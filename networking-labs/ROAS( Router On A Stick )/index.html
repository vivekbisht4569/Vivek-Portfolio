Router On A Stick(ROAS)

Definition

Router-on-a-Stick is a method used to enable communication between multiple VLANs using a single physical router interface. It is called “stick” because only one cable (one interface) connects the router to the switch.

How It Works

1️⃣Switch Side The switch port connected to router is configured as: Config: switchport mode trunk This trunk carries traffic of multiple VLANs. 2️⃣ Router Side Instead of using multiple physical interfaces, we create: 👉 Subinterfaces Example: interface g0/0.10 encapsulation dot1Q 10 ip address 192.168.10.1 255.255.255.0 interface g0/0.20 encapsulation dot1Q 20 ip address 192.168.20.1 255.255.255.0 Each subinterface: Belongs to one VLAN Has one gateway IP Routes traffic between VLANs

Why we need it ?

By default: Devices in same VLAN → can communicate ✅ Devices in different VLANs → cannot communicate ❌ To allow different VLANs to talk to each other, we need: 👉 Inter-VLAN Routing Router-on-a-Stick is one way to do that.
