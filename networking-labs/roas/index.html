Router On A Stick (ROAS)

Definition

Router-on-a-Stick (ROAS) is a method used to enable communication between multiple VLANs using a single physical router interface.

It is called “stick” because only one physical cable connects the router to the switch.

How It Works

1️⃣ Switch Side

The switch port connected to the router is configured as a trunk port. 

switchport mode trunk

This trunk link carries traffic from multiple VLANs to the router.

2️⃣ Router Side

Instead of using multiple physical interfaces, we create logical subinterfaces on a single router interface.

Example configuration:

interface g0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0

interface g0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0
  • Each subinterface belongs to one VLAN.
  • Each subinterface has its own default gateway IP.
  • The router routes traffic between VLANs.

Why We Need It

By default:

  • Devices in the same VLAN → can communicate ✅
  • Devices in different VLANs → cannot communicate ❌

To allow communication between different VLANs, we need Inter-VLAN Routing.

Router-on-a-Stick is one common method to achieve Inter-VLAN Routing.
